Adaptive Reading Check (ARC)

A computer-adaptive reading placement for ESL learners.
为英语学习者设计的自适应阅读测评。

Show Chinese task instructions to student / 向学生显示中文题目说明
Integrity mode (block copy, log tab-switches) / 诚信模式
This is an original classroom tool. It uses the public-domain Rasch adaptive model — the same science behind well-known adaptive tests — but contains no proprietary questions, scales, or norms from any commercial product. Scores are calibrated to this item pool, useful for grouping and for tracking growth when re-administered. They are not nationally normed grade levels.

How this works / 测试说明

1. Read each question and choose the best answer.

3. You cannot go back. Take your time and pick carefully.

没有时间限制，尽力而为！
